Features

Everything you need to run sites — nothing you don't.

Qadbak ships with the workflows hosting providers actually use, in a UI that doesn't feel like 2008.

Domains

Add sites in seconds — subdomains, aliases, redirects and reverse proxies. Per-site nginx, lifecycle in one list.

Email

Mailboxes, forwarding, IMAP, mail logs and a sane SPF/DKIM setup helper. Scoped per domain for clients.

DNS & SSL

One-click DNS records, automatic Let's Encrypt, and a clear status badge per certificate.

Databases

MariaDB databases and users managed per-domain, with phpMyAdmin shortcut when you need it.

Files & terminal

Native file manager + in-browser shell as the domain unix user. No more dragging FTP clients around.

Server admin

Services, stack reload, backups, resellers, system metrics — everything an operator needs.

Deploy

Three steps from cold VPS to live panel.

No vendor lock-in, no proprietary stack. Vanilla nginx + Postfix + MariaDB on Ubuntu 22.04 / 24.04.

01

Provision a VPS

Any €5/month Ubuntu 22.04 or 24.04 box works. Point an A record (your-host.com) to the IP.

02

Run the installer

sudo bash install/qadbak-install.sh from github.com/macdirtycow/qadbak. Sets up nginx, mail, MariaDB, panel.

03

Activate Premium

Sign in → Server admin → License → paste your QAD-… key → Refresh modules. Done.
